- Jeśli nie masz Docker Desktop, to pobierz aplikacje Docker Desktop z https://www.docker.com/products/docker-desktop/
- Uruchom Docker Desktop i w terminalu w prawym dolnym rogu wpisz: docker run -d --name dachshundDB -p 27017:27017 mongo:6.0
- Aby uruchomic kolejny raz użyj: docker start dachshundDB

nazwa bazy: dachshundDB
Port w kontenerze: 27017
Port na komputerze: 27017
użyty silnik: mongo6.0
